# GitBash 學習筆記

### 如何安裝GitBash

https://git-scm.com/download/win

### Git 有哪些用途

- 版本控制系統

- 遊戲存檔功能

- 共同開發

### 什麼是GitHub?
- 遠端hosting，讓大家方便使用(可以自己host)
- 跟google drive很像
### 常用指令
1. git add
2. git commit 
3. git push
4. git remote
5. git status

## 如何將檔案上傳到GitHub
1. git init 
將新建的資料夾初始化
2. git status 查詢狀態
3. git add .
4. git commit
5. q!
